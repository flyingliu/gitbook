# 场景分组插件

场景分组方法

方法名 |  参数 |  作用
-----|----|----
addScene | String scene | 添加场景
getCurrScene | 无 | 获取当前场景
getGroup | 无 | 获取场景分组
getScene | String name | 获取场景列表
loadScene | String name | 加载场景
removeScene | String scene | 删除场景
onnewscene | 无 | 加载场景触发方法
init | 无 | 初始化插件

调用例子：

    yp.plugin.group.getGroup()

获取场景分组
